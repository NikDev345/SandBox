    Common Issues We Fix

    A sample of JSON patterns this tool automatically detects and corrects.

    Trailing commas

    {"a": 1,} → {"a": 1}

    Single quotes

    {'a': 1} → {"a": 1}

    Unquoted keys

    {a: 1} → {"a": 1}

    Missing commas

    {"a": 1 "b": 2} → {"a": 1, "b": 2}

    Comments

    {"a": 1} // note → {"a": 1}

    Truncated JSON

    {"a": [1, 2 → {"a": [1, 2]}
